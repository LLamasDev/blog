## Instalar vi

```bash
apt-get install vim
```

## Abrir

```bash
vi ALGO
```

## Insertar

```bash
i
```

## Buscar

```bash
/ALGO
```

## Primera línea

```bash
1G
```

## Ir a una línea

Ir a la línea 11:
```bash
11G
```

## Última línea

```bash
G
```

## Guardar

```bash
:w
```

Guardar y salir:
```bash
:wq
```

## Salir

```bash
:q
```

Salir sin guardar:
```bash
:q!
```
