## Instalar

```bash
apt-get install cowsay
```

## ¿Cómo usarlo?

![image-center]({{ site.url }}{{ site.baseurl }}./assets/img/TuxBlog.png){: .align-center}

Vaca:
```bash
cowsay ALGO
```

Cambiar icono:  
`stegosaurus`  
`tux`  
`ghostbusters`
```bash
cowsay -f ICONO ALGO
```
