## Buscar para asegurarte

```bash
find . -iname 'ALGO' 2>/dev/null -exec ls -l {} \;
```

## Copiar

```bash
find . -iname 'ALGO' 2>/dev/null -exec cp {} /tmp/CARPETA/ \;
```
